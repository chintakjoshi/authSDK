# Documentation Hub

This hub is the recommended starting point for new engineers.

## Pick Your Path

- I need to run the auth service locally:
  read `../DEVELOPMENT.md`
- I need to integrate this SDK into my app:
  read `integrate-sdk.md`
- I need endpoint contracts and payload examples:
  read `service-api.md`
- I need production runtime and operational notes:
  read `operations.md`
- I am blocked and need fast debugging help:
  read `troubleshooting.md`
- I need security review artifacts:
  read `security-review.md`

## Suggested First-Day Journey

1. Run the stack locally with `../DEVELOPMENT.md`.
2. Complete `integrate-sdk.md` and protect one route in your app.
3. Validate your setup using curl examples in `service-api.md`.
4. If anything fails, use `troubleshooting.md`.
